# Playnix

Playnix is the foundation for a premium gaming marketplace inspired by the operational model we analyzed, but branded and designed around a custom Playnix identity.

## Current Scope

- Next.js application shell
- Shared Playnix design system and dark theme
- Landing page with core marketplace positioning
- Placeholder routes for marketplace, selling, and support
- Architecture notes for backend and data model planning

## Commands

```bash
npm install
npm run dev
npm run build
npm run typecheck
```

## Next Build Phases

1. Authentication and user onboarding
2. Marketplace browse and offer detail pages
3. Seller dashboard and listing management
4. Orders, chat, disputes, and warranty flows
5. Payments, wallet, payouts, and admin operations
