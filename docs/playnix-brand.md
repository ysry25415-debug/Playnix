# Playnix Brand Palette

Playnix uses a dark, premium, cyberpunk-inspired palette built around a bluish black foundation and aggressive neon red accents.

## Core Backgrounds

| Token | HEX | Usage |
| --- | --- | --- |
| `bg-canvas` | `#05070B` | Main app background |
| `bg-elevated` | `#0A0F16` | Secondary sections and panels |
| `bg-surface` | `#101722` | Cards, forms, drawers |
| `bg-surface-strong` | `#161F2D` | Hovered cards, active containers |
| `bg-inset` | `#1B2636` | Deep nested containers and control wells |

## Primary Red Scale

| Token | HEX | Usage |
| --- | --- | --- |
| `red-900` | `#42060D` | Deep shadows, destructive depth |
| `red-800` | `#6E0B16` | Dark accent surfaces |
| `red-700` | `#A50F20` | Borders, pressed actions |
| `red-600` | `#D81629` | Primary hover state |
| `red-500` | `#FF2A2A` | Primary buttons, key CTAs |
| `red-400` | `#FF5A5A` | Bright highlights, focused accents |

## Text

| Token | HEX | Usage |

| `text-on-accent` | `#FFF5F5` | Text on red buttons |

| Semantic Token | Value | Usage |
| --- | --- | --- |
| `button-primary-bg` | `#FF2A2A` | Main action buttons |
| `button-primary-hover` | `#D81629` | Hover state for primary buttons |
| `button-primary-text` | `#FFF5F5` | Button text |

| `focus-ring` | `#FF434380` | Accessible focus glow |
| `danger-surface` | `#42060D` | Critical alerts |

## Notes

- The palette is optimized for dark mode first.
- Red stays intense without washing out the UI.
- Light neutrals are kept soft to preserve the premium gaming tone.
- Bright or pastel support colors should be avoided unless we intentionally add a success/warning system later.
